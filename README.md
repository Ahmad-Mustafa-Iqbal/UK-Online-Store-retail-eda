# Online Retail Store - Exploratory Data Analysis (EDA)

## Project Overview
This repository contains a comprehensive Exploratory Data Analysis (EDA) of a UK-based online retail company. The objective of this project is to understand customer purchase behaviors, identify data quality issues, and discover key business insights before investing in a recommendation system.

This task was completed as part of Week 1 of my AI/ML Internship.

## Dataset Information
* **Name:** Online Retail II Dataset
* **Source:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets/Online+Retail+II)
* **Description:** The dataset contains real transaction data from 2009 to 2011, including invoices, product descriptions, quantities, prices, and customer IDs.

## Tech Stack Used
* **Language:** Python
* **Libraries:** `pandas`, `numpy`, `matplotlib`, `seaborn`
* **Environment:** Jupyter Notebook / Google Colab

## Key Tasks Performed
1. **Data Loading & Cleaning:** Checked for missing values, duplicates, and correct data types without removing any observations.
2. **Sales & Revenue Analysis:** Calculated total revenue and identified the top 10 best-selling products by quantity and revenue.
3. **Geographical Analysis:** Evaluated sales performance across different countries.
4. **Visualizations:**
   - Monthly revenue trend analysis (Line Plot).
   - Outlier detection in Quantity and Price (Box Plots).
   - Feature correlation (Heatmap).
5. **Business Insights:** Derived top 5 actionable insights for business growth.
